# MIM Games
This is a Boardgames E-Commerce store project.


##Technologies
* Project created with:
* Node Package Manager: 8.5.0
* React App: 18.0.0
* React Icons: 4.3.1

### Setup
To run this project install it locally using npm:

$ cd ../mimgames-mmiranda
$ npm install
$ npm install react-icons --save
$ npm start
